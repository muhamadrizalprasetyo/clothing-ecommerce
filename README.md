# Cozzy Store - Frontend

Premium Minimalist Streetwear E-commerce Storefront.

## Tech Stack
- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **State Management**: Context API

## Features
- Elite Industrial-Minimalist UI
- Premium Editorial Footer & Navbar
- Responsive Design
- Optimized for Performance

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
